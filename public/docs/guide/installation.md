# 安装指南

本文档介绍如何在本地启动 Lyra Tavern。

## 前置要求

- Node.js 18+
- pnpm / npm / yarn

## 步骤

1. 克隆仓库

```bash
git clone https://github.com/yourname/lyra-tavern.git
```

2. 安装依赖

```bash
npm install
```

3. 启动开发服务器

```bash
npm run dev
```

访问 `http://localhost:3000` 查看效果。
