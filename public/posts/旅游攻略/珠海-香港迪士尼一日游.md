# 🚗 一、出行前准备：让旅程更顺畅 ✅

## 🔹 必备物品清单 🧳

- 港澳通行证 + 签注 ✅
- 香港流量包（推荐提前购买）📶📱
- 充电宝 + 手机（拍照用！📸）
- 舒适鞋子👟 + 帽子👒 + 防晒霜🧴
- 小零食（不能带自热食品🚫）
- 水杯（园区免费接水💧）

## 🔹 提前购票提醒 ⚠️

✅ **门票**：提前在官网或官方APP购买，避免现场排队。

✅ **穿梭巴士票**：可通过“港珠澳大桥穿梭巴士”公众号线上购票，支持微信/支付宝支付。

✅ **尊享卡/提早入园证**：强烈推荐！

> 💡 小贴士：节假日人流大，建议至少提前1周购票！

# 🚌 二、珠海 → 香港：穿越大桥的浪漫之旅 🌉

## 🕒 出发时间建议：⏰ 7:00–7:30

早起是成功的关键！🌅 如果你是自驾游客，可将车停在**珠海口岸P2停车场**（约¥10/小时），非自驾则打车前往即可🚕。

## 🛑 珠海口岸通关流程

1. 进入口岸后走**夜间通道**（M层）🚪
2. 准备好港澳通行证，依次完成：
   - 刷卡 ➜ 刷脸 ➜ 刷指纹 ✅
3. 安检后前往**金巴候车区**

## 🚌 乘坐穿梭巴士（金巴）🚌

- **票价**：💰 ¥58/人（日间）｜¥63/人（夜间）
- **乘车点**：珠海口岸地下一层
- **班次密集**：每15分钟一班，24小时运营
- **上车前务必换取纸质票**❗（线上购票需取票）

🎯 **最佳座位推荐**：二楼右侧靠窗位，欣赏港珠澳大桥海景🌊📸

# 🌍 三、抵达香港 → 前往迪士尼 🎪

## 🚌 B5路巴士直达方案（最推荐）✅

- **站点**：港珠澳大桥香港口岸 → 迪士尼站
- **票价**：HK$6.1/人（注意：截图中“HK 7.4”可能是误写或包含其他费用）
- **时间**：约15分钟
- **注意**：只有特定时段直达，其余需换乘地铁

## 📅 B5直达班次（仅供参考）：

|     时间      |     是否直达     |
| :-----------: | :--------------: |
| 8:07 – 11:07  |      ✅ 直达      |
| 20:00 – 22:16 |      ✅ 直达      |
|   其他时段    | ❌ 需在欣澳站换乘 |

> 🔄 若错过直达班次，请在**欣澳站A出口**步行至B5站台，换乘公交前往迪士尼（总耗时约45分钟）

# 🍽️ 八、餐饮建议：吃得开心又省钱 🍔🥤

# 🌅 九、返程指南：从迪士尼回到现实 🚑

## ✅ 最优方案：B5直达 + 穿梭巴士

1. 在**迪士尼站**乘坐B5直达港珠澳大桥香港口岸（HK$6.1/人，约15分钟）
2. 在口岸转乘穿梭巴士返回珠海（¥58/人，约45分钟）

> 🕒 B5末班车时间：22:16（周末及节假日可能延长）

## 🚇 备选方案：地铁 + 公交

- 迪士尼 → 欣澳站（HK$7.4）→ B5公交 → 珠海

## 🚕 氪金方案：出租车

- 约HK$180，行李多时额外收费，适合携带大量行李的家庭

# 💡 十、旅行小贴士 🌟

1. 🕰️ **早入园 = 少排队**：尽量在开园前抵达
2. 📱 **下载“香港迪士尼乐园”APP**：实时查看排队时间、演出安排
3. 🧺 **储物柜**：园区内有付费储物柜（约HK$20/次）
4. 🎒 **轻装上阵**：只带必需品，减轻负担
5. 🎵 **与角色合影**：记得微笑、摆pose，留下美好回忆！📷
6. 🌧️ **天气变化快**：随身携带雨衣或折叠伞
